# Stage 1B - Profile Card

This is a simple responsive Profile Card built as part of Stage 1B frontend task.  
It displays user information, social links, hobbies, dislikes, and a live updating timestamp.


## Live Demo
https://profile-card-lilac-zeta.vercel.app/

## Project Structure
- index.html
- style.css
- script.js


## Features
- Responsive profile card layout
- Semantic HTML structure
- Live time display using `Date.now()`
- Social media links (opens in new tab)
- Hobbies and dislikes lists
- Accessible image with alt text
- Keyboard-friendly navigation


## How it works
- The time is generated using `Date.now()` and updates every second
- Layout adjusts for mobile and desktop screens using CSS flexbox and media queries


## How to Run Locally
Simply open the `index.html` file in any modern browser.
